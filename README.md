## プロジェクトの目的
共通の目的を持った人が集まるきっかけづくりの場を提供する
## 使用方法
「目的」を持った人が店の情報や、店の場所、名前等から開催する店で部屋を立てる。  
次にその「目的」に興味を持った人がその部屋の情報（開催日や場所など）を確認し、気に入ればその部屋へ入る。  
開催日までに部屋の中で確認してもらい、当日に集合場所へ集まって食事、交流をしていただく。
